# Trial

At the moment, we do not provide a portal for requesting a trial license. Instead, you can request one through the CodeGlass application. The steps below explain how to register for a trial.

For more details on setting up CodeGlass on your computer, follow the instructions in [Getting Started](./category/getting-started) up to the [Register](./views/general/register) page.

1. Download the **latest release** from the [CodeGlass GitHub repository](https://github.com/CodeGlassDotIO/CodeGlass).
2. Extract the downloaded archive to a location on your system.
3. Navigate to the folder where you extracted CodeGlass.
4. Start the [CodeGlass.Engine](intro#engine).

This will start the CodeGlass background services and will try to open a browser to the [CodeGlass Client](intro#client).

:::info
If the browser does not open automatically, you can manually navigate to:
```bash
http://localhost:30340
```
:::

5. Click the _Register_ button on the [login](./views/general/login) page.
6. Enter your email address and click _Register_.
7. Return to the login page by clicking _Log In_.
8. Enter your registered email address and the password sent to that email.
9. Start using CodeGlass!

If you experience any issues, please feel free to [contact us](../contact).
