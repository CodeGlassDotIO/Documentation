# About CodeGlass

CodeGlass develops tools for inspecting and understanding the runtime behavior of applications. The current focus is on the **CodeGlass for Julia** program, where we work with teams using Julia to help improve the development tooling available in the ecosystem.

If your organization is using Julia and is interested in participating in this program, feel free to [reach out](/contact).
